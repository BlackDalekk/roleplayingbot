# Hero

### Социальные атрибуты
* Имя
* Описание
* Биография
* Раса
* Статус (Крестьянин, Дворянин и проч. меняется отношение и взаимодействие(зарплата например))
* Убеждение (Статус должен влиять)
* Знание легенд (Опознание)
* Скрытность (должно зависить расы + на него влияет экипировка)
* Воровство (скрытность скрытнотсь) -> навык

### Игровые атрибуты
* Опыт -> Уровень
* HP (константно, ведь броня не увеличивает его. Фактически это тупо кол-во крови, после потери который ты дохнешь)
* Мана
* Резисты:
  * Физ
  * Маг-стихийное
  * Стихийной
  * Маг чист
  * Сила
  * Ловкость
  * Интеллект
  * Выносливость
> При блокировке урона (5 брони, 4 урона - теряешь выносливость. 5 брони 6 урона - теряешь и выносливость и хп)
